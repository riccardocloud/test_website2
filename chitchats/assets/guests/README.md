Drop guest portraits here. Recommended: square JPG/WebP, 400×400 or larger.

Filename convention: `firstname-lastname.jpg` (lowercase, hyphens).

To use one, replace a monogram `.guest-avatar` in `index.html`:

```html
<div class="guest-avatar" style="background:#18578F">RM</div>
<!-- becomes -->
<img class="guest-avatar" src="assets/guests/riccardo-murciano.jpg" alt="Riccardo Murciano">
```
